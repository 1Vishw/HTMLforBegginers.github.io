Basics of HTML

What is HTML:

HTML, or HyperText Markup Language, is the standard markup language used to create and design web pages. It provides a basic structure for content on the World Wide Web, allowing developers to define and organize the various elements that make up a webpage. Here's a brief overview of the basics of HTML

HTML Document Structure:

An HTML document is structured using tags, which are enclosed in angle brackets (< >). The basic structure of an HTML document includes:

  • !DOCTYPE html: Declares the document type and version of HTML (HTML5 in this case).
  • html: The root element that wraps the entire HTML content.
  • head: Contains meta-information about the document, such as the title and links to stylesheets.
  • title: Sets the title of the HTML document, which is displayed in the browser's title bar or tab.
  • body Contains the main content of the HTML document.

HTML Tags

  • Paragraph tag:

The paragraph tag is used to define paragraphs in HTML. Its primary purpose is to structure and separate blocks of text on a webpage. When you use the paragraph tag, the browser automatically adds spacing before and after the paragraph, making it visually distinct from other content.
